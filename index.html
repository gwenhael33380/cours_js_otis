Les variables

Définition d'une variable

C'est un espace stockage qui doit être déclaré avec un nom, et que l'on peut à tout moment modifier, on attribue la valeur à une variable avec le signe '='

Format du nom d'une variable en JS

  • La variable est sensible à la casse : variable, Variable, vArIaBlE sont 3 variables distinctes
  • La variable ne peut etre composé que de lettre, de chiffre et pas de caractères spéciaux (sauf _ et $)
  • La variable ne peut pas commencer par un chiffre
  • On ne doit pas utiliser comme nom de variable une fonction JS connu (var, let, return, break ...)
  • Par convention, on va nommer les variables JS en utilisant le camelCase

Déclaration d'une variable

On peut déclarer une variable avec let ou var

La différence principale est leur portée, mais nous verrons cela plus tard après avoir étudié les conditions, les boucles et les fonctions

L'autre différence est qu'avec let on ne peut pas déclarer 2 fois la même variable (dans le même scope)

Le type de données d'une variable

  • string ==> chaîne de caractères
  • Number ==> chiffres
  • Boolean ==> true or false
  • null
  • undefined
  • NaN ==> Not a Number
  • Symbol ==> Array ou tableau
  • Object

Les opérateurs de comparaison

  • == pour voir si 2 valeurs (ou variables) sont égales
  • === pour voir si 2 valeurs (ou variables) sont strictement égales
  • != pour voir si 2 valeurs sont différentes
  • !== pour voir si 2 valeurs sont strictement différentes
  • > plus grand que
  • >= supérieur ou égal
  • < plus petit que
  • <= inférieur ou égal

La concaténation

C'est le fait d'associer deux éléments ensembles (variables, string, integer)

Les constantes

C'est également un espace de stockage, qui peut avoir les même types de données qu'une variable

Elle a les même restrictions de nommage qu'une variable

Celle-ci sera déclarée avec const

La seule différence avec une variable, c'est qu'une fois déclarée, on ne pourra plus la modifier

Les sélecteurs d'éléments

Pour attacher une fonction JS à un élément, il va falloir, dans un premier temps le sélectionner

  • document.getElementsByTagName(localName: DOMString); va sélectionner les balises
  • document.getElementById(elementId: DOMString); va sélectionner le premier id correspondant
  • document.getElementsByClassName(classNames: DOMString); va sélectionne toutes les classes css correspondantes
  • document.querySelector(selectors: DOMString); va sélectionner le premier élément trouvé, il va falloir lui indiquer si c'est une classe ou un id, on pourra utiliser les sélecteurs d'élements
  • document.querySelectorAll(selectors: DOMString) va sélectionner tous les élements ciblés dans le DOM, de la même façon que querySelector, il faudra indiquer si c'est une classe '.' ou un id '#'

Les conditions

Les conditions if else

le 'if' va interpréter une condition, si celle-ci est valide, elle va exécuter une instruction

si un else est présent, et que la condition du if n'est pas valide, c'est l'instruction du else qui va être exécuté

Les conditions if ... else if ... else

Celle-ci fonctionne de la même façon que le if else, mais dans le else if on va pouvoir rajouter des conditions supplémentaires

On peut mettre autant de else if que l'on veut

Les opérateurs logiques

  • && ==> correspond à 'et'
  • || ==> correspond à 'ou'

L'opérateur ternaire

c'est une condition simple dont le résultat var alimentater directement la variable

La condition switch ... case

L'instruction switch évalue une expression et, selon le résultat obtenu et le cas associé, exécute les instructions correspondantes.

Les boucles

La boucle for

Pour fonctionner cette boucle a besoin d'un point de départ, un point d'arrivée et une itération pour chaque tour de boucle

for(var i = 1; i <= 10; i++)

Dans cet exemple il y aura 10 tour de boucle
